# Gaming Web Store 

**Live Demo:** [https://maxi-rv.github.io/gaming-store](https://maxi-rv.github.io/gaming-store)

A responsive static e-commerce store built with vanilla HTML, CSS (Utilizing Bootstrap 5.3), and JavaScript.

This project is a forked continuation of an introductory 1-month university project. 

Originally developed by [dariopokorny](https://github.com/dariopokorny) and [maxi-rv](https://github.com/maxi-rv) 

## Overview

A lightweight, static e-commerce storefront built entirely with vanilla web technologies. It provides a modern shopping experience, making it perfect for small businesses, portfolios, or learning purposes.

### 🎨 Design
- Fully responsive layout
- Modern, clean UI with Bootstrap + PopperJS
- Dark theme

### 🛒 Shopping Experience
- Product catalog with grid view
- Advanced filtering by category, tags, stock
- Shopping cart with add/remove/update quantities

### 💻 Technical
- **HTML5** - Semantic markup
- **CSS** - Custom properties, Flexbox, Grid, Animations
- **JavaScript ES6+** - Vanilla JS, DOM manipulation, Events
- **Bootstrap Icons** - Icon library
- **LocalStorage** - Client-side data persistence + Mocking backend functionality
